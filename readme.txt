This project allows a user to interact with a WebGL drawing canvas.
You can either upload a predefined .dat file to draw that image to the canvas, or enter drawing mode to click and draw points.

--- How to use it ---
Open up "index.html" to view and interact with the program
Upload a .dat file from the "dat files folder"

--- File Mode ---
Enter file mode by pressing 'f'
Cycle between colors by pressing 'c'

--- Drawing Mode ---
Enter drawing mode by pressing 'd'
Click to draw a new point, and each additional click will draw a line
Hold 'b' and click to start a new line
Cycle between colors by pressing 'c'
